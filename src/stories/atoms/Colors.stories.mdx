# Colors
The library's color palette is designed to meet [WCAG 2 Guidelines](https://www.w3.org/TR/WCAG20/#visual-audio-contrast-contrast) for [minimum color contrast](https://dequeuniversity.com/rules/axe/4.4/color-contrast?application=axeAPI) to font color:

| Theme | Color | Aa - Large Text | Aa - Normal Text | Contrast Ratio |
|---|---|---|---|---|
| Light | <div style={{ backgroundColor: colors.light.turquoise[100] }} /> `turquoise-100` | Black Text - min 49% opacity | Black Text - min 65% opacity | [8.05:1](https://webaim.org/resources/contrastchecker/?fcolor=000000&bcolor=00B0C9) |
| Light | <div style={{ backgroundColor: colors.light.gold[100] }} /> `gold-100` | Black Text - min 46% opacity | Black Text - min 60% opacity | [10.49:1](https://webaim.org/resources/contrastchecker/?fcolor=000000&bcolor=FFA300) |
| Light | <div style={{ backgroundColor: colors.light['hot-pink'][100] }} /> `hot-pink-100` | Black Text - min 55% opacity | Black Text - min 77% opacity | [5.49:1](https://webaim.org/resources/contrastchecker/?fcolor=000000&bcolor=FF1654) |
| Dark | <div style={{ backgroundColor: colors.dark.turquoise[100] }} /> `turquoise-100` | White Text - min 70% opacity | White Text - min 100% opacity | [4.51:1](https://webaim.org/resources/contrastchecker/?fcolor=FFFFFF&bcolor=008299) |
| Dark | <div style={{ backgroundColor: colors.dark.gold[100] }} /> `gold-100` | White Text - min 64% opacity | White Text - min 92% opacity | [5:1](https://webaim.org/resources/contrastchecker/?fcolor=FFFFFF&bcolor=A35F00) |\
| Dark | <div style={{ backgroundColor: colors.dark['hot-pink'][100] }} /> `hot-pink-100` | White Text - min 63% opacity | White Text - min 83% opacity | [6.2:1](https://webaim.org/resources/contrastchecker/?fcolor=FFFFFF&bcolor=C4002C) |
